## 📖 Concepts Covered

### 1. Simple Function

Created a function using the `def` keyword.

Example:

```python
def greet():
    print("Hello")
```

---

### 2. Calling a Function

Executed a function by writing its name followed by parentheses.

Example:

```python
greet()
```

---

### 3. Function with Parameters

Created a function that accepts data while calling it.

Example:

```python
def greet_user(name):
    print("Hello", name)
```

---

### 4. Function Returning a Value

Learned that a function can return a value using the `return` keyword.

Example:

```python
def add(a, b):
    return a + b
```

## 📚 What I Learned Today

* A function is a reusable block of code.
* `def` is used to define a function.
* A function runs only when it is called.
* Parameters allow the same function to work with different values.
* `return` sends a value back to the caller.
